# Operações Básicas Map
Esta seção do projeto é destinada à prática das operações básicas envolvendo a interface Map em Java. Isso foi feito por meio da criação de uma agenda de contatos com os métodos:
- adicionarContato()
- removerContato()
- exibirContatos()
- pesquisarPorNome()

## Aprendizados
Por meio dessas funções, foi possível entender a dinâmica de armazenamento por conjuntos key/value utilizados por essa interface.
