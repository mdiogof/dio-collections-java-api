# Pesquisa com Map
Esta seção do projeto é destinada à prática da pesquisa de elementos de maps em Java. Isso foi feito por meio da criação de um estoque de produtos com os métodos:
- adicionarProduto()
- exibirProdutos()
- calcularValorTotalEstoque()
- obterProdutoMaisCaro()
- obterProdutoMaisBarato()
- obterProdutoMaiorQuantidadeValorTotalNoEstoque()

## Aprendizados
Por meio dessas funções, foi possível praticar o gerenciamento de informações utilizando a interface Map. Além disso, compreendi o uso da interface `Map.Entry<>` para ampliar o acesso aos valores contidos em uma Map.
