# Ordenação com Map
Esta seção do projeto é destinada à prática da ordenação de elementos de maps em Java. Isso foi feito por meio da criação de uma agenda de eventos com os métodos:
- adicionarEvento()
- exibirAgenda()
- obterProximoEvento()

## Aprendizados
Por meio dessas funções, foi possível praticar a ordenação automática de elementos com a interface Map. Além disso, entendi o funcionamento de alguns métodos disponibilizados pela classe `LocalDate`, a qual foi fundamental para organizar a agenda de eventos criada.
