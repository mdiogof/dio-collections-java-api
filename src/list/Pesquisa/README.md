# Pesquisa com List
Esta seção do projeto é destinada à prática da pesquisa de elementos de lists em Java. Isso foi feito por meio da criação de um catálogo de livros com os métodos:
- adicionarLivro()
- pesquisarPorAutor()
- pesquisarPorIntervaloAnos()
- pesquisarPorTitulo()

E de uma lista de números com os métodos:
- adicionarNumero()
- calcularSoma()
- encontrarMaiorNumero()
- encontrarMenorNumero()
- exibirNumeros()

## Aprendizados
Por meio dessas funções, foi possível praticar o uso de laços `for` e laços condicionais em lists para encontrar os elementos desejados.
