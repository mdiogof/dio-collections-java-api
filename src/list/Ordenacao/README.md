# Ordenação com List
Esta seção do projeto é destinada à prática da ordenação de lists em Java. Isso foi feito por meio da criação de uma lista de pessoas com os métodos:
- adicionarTarefa()
- ordenarPorIdade()
- ordenarPorAltura()

E de uma lista de números com os métodos:
- adicionarNumero()
- ordenarAscendente()
- ordenarDescendente()
- exibirNumeros()

## Aprendizados
Por meio dessas funções, foi possível praticar o uso das interfaces `Comparable` e `Comparator`.
