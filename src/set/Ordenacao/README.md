# Ordenação com Set
Esta seção do projeto é destinada à prática da ordenação de sets em Java. Isso foi feito por meio da criação de uma lista de produtos com os métodos:
- adicionarProduto()
- exibirProdutoPorNome()
- exibirPorPreco()

## Aprendizados
Por meio dessas funções, foi possível praticar o uso da implementação `TreeSet` em conjunto com as interfaces Comparable e Comparator para organizar listas.
