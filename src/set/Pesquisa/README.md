# Ordenação com List
Esta seção do projeto é destinada à prática da pesquisa de elementos de sets em Java. Isso foi feito por meio da criação de uma agenda de contatos com os métodos:
- adicionarContato()
- exibirContatos()
- pesquisarPorNome()
- atualizarNumeroContato()

## Aprendizados
Por meio dessas funções, foi possível praticar o uso de laços `for` e laços condicionais em sets para encontrar os elementos desejados.
