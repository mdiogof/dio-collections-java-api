# Operações Básicas Set
Esta seção do projeto é destinada à prática das operações básicas com sets em Java. Isso foi feito por meio da criação de uma lista de convidados com os métodos:
- adicionarConvidado()
- removerConvidado()
- contarConvidados()
- exibirConvidados()

## Aprendizados
Por meio dessas funções, foi possível entender o funcionamento da collection `Set` e da sua implementação `HashSet`, evitando a repetiçao de elementos com um atributo especificado em comum.
